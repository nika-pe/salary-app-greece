Υπολογιστής Μισθού στην Ελλάδα
Περιγραφή
Αυτό είναι ένα απλό web-application που αναπτύχθηκε με Flask, το οποίο επιτρέπει στους χρήστες να υπολογίζουν τον καθαρό μισθό τους στην Ελλάδα με βάση τον μικτό μισθό και την κατηγορία εργαζομένου. Η εφαρμογή υποστηρίζει επίσης την εξαγωγή αποτελεσμάτων σε Excel, τη φόρτωση αρχείων (Excel/PDF) για επεξεργασία δεδομένων και τη δημιουργία συνοπτικών αναφορών.

Χαρακτηριστικά
Υπολογισμός καθαρού μισθού με βάση τον μικτό μισθό και την κατηγορία εργαζομένου (μισθωτός, ελεύθερος επαγγελματίας, δημόσιος υπάλληλος).

Εξαγωγή αποτελεσμάτων υπολογισμού σε αρχείο Excel.

Φόρτωση αρχείων Excel/PDF για περαιτέρω επεξεργασία δεδομένων.

Δημιουργία συνοπτικών αναφορών (προς το παρόν με δείγμα δεδομένων).

Πλήρως μεταφρασμένο περιβάλλον χρήστη στα Ελληνικά.

Τεχνολογίες που χρησιμοποιήθηκαν
Python

Flask: Web framework

Jinja2: Templating engine

Tailwind CSS: For styling and responsive design

Pandas: For data manipulation and Excel handling

Openpyxl: For Excel file operations

Werkzeug: For secure file uploads

Εγκατάσταση και Εκτέλεση
Προαπαιτούμενα
Python 3.x εγκατεστημένο.

Microsoft Visual C++ Build Tools (απαιτείται για την εγκατάσταση ορισμένων βιβλιοθηκών όπως το Pandas/Numpy σε Windows).

Βήματα
Κλωνοποιήστε το αποθετήριο:

git clone https://openedtech.ellak.gr/github-instructions/
cd salary_project


Δημιουργήστε ένα εικονικό περιβάλλον (συνιστάται):

python -m venv venv


Ενεργοποιήστε το εικονικό περιβάλλον:

Windows:

.\venv\Scripts\activate


macOS/Linux:

source venv/bin/activate


Εγκαταστήστε τις εξαρτήσεις:

pip install -r requirements.txt


Εκτελέστε την εφαρμογή:

python app.py


Ανοίξτε την εφαρμογή στο πρόγραμμα περιήγησης:
Μεταβείτε στη διεύθυνση http://127.0.0.1:5000

Χρήση
Εισάγετε τον μικτό μισθό και επιλέξτε την κατηγορία εργαζομένου για να δείτε τον καθαρό μισθό, το ΙΚΑ και τον φόρο.

Χρησιμοποιήστε το κουμπί "Αποθήκευση σε Excel" για να κατεβάσετε τα αποτελέσματα του υπολογισμού.

Φορτώστε αρχεία Excel ή PDF χρησιμοποιώντας την αντίστοιχη φόρμα.

Πατήστε "Δημιουργία Αναφορών" για να δείτε μια συνοπτική αναφορά (δείγμα).

Πιθανές Βελτιώσεις
Ενσωμάτωση βάσης δεδομένων για αποθήκευση ιστορικού υπολογισμών και δεδομένων φόρτωσης.

Πιο προηγμένη επεξεργασία αρχείων PDF.

Ανανέωση αποτελεσμάτων σε πραγματικό χρόνο χωρίς ανανέωση σελίδας (με AJAX/JavaScript).

Προσθήκη περισσότερων κατηγοριών εργαζομένων ή παραμέτρων υπολογισμού.
